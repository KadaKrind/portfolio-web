# Portfolio - Rodrigo Porcel (Full Stack Developer)

## Project Description
A professional portfolio website showcasing projects, skills, experience, and contact information for a full stack web developer. Features project filtering, skills proficiency display, timeline of work experience, and a contact form.

## Features
- **Hero Section**: Compelling introduction with call-to-action
- **About Section**: Professional bio and key statistics
- **Portfolio**: 6+ project showcases with filtering by category
- **Skills Section**: Technical skills organized by category with proficiency levels
- **Experience Timeline**: Chronological view of work experience
- **Contact Form**: Direct messaging capability
- **Responsive Design**: Mobile-friendly layout
- **Social Links**: Links to GitHub, LinkedIn, Twitter, Dribbble
- **Professional UI**: Modern design with Font Awesome icons

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, flexbox, animations, gradients
- **JavaScript (ES6+)**: Project filtering, smooth scrolling, form handling
- **Font Awesome 6.4**: Professional icon library (CDN)
- **No Dependencies**: Pure vanilla JavaScript

## Key Features Implemented

### Portfolio Section
- 6 sample projects with images, descriptions, and tech stacks

- Project cards with hover effects

### Skills Section

### Experience Section

- Key statistics (projects, clients, years)

### Contact Section
- Contact form with validation
- Contact information with icons

### Responsive Design
- Mobile-friendly layout
- Hamburger menu for mobile navigation
- Adaptive grid system
- Touch-friendly buttons and controls

## How to Use
1. Open `index.html` in a web browser
2. Navigate through different sections using the sticky header
3. Browse featured projects and filter by category
4. Review technical skills and proficiency levels
5. Check out work experience timeline
6. Fill out contact form to send a message
7. Click social links to connect

## Customization
- Replace project data in the `projects` array in `js/script.js`
- Update skills and proficiency levels in HTML
- Modify timeline entries for your experience
- Update contact information with real details
- Add real social media links
