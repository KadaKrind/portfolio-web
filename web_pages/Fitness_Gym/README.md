# Fitness/Gym Website - FitZone

## Project Description
A professional fitness gym website showcasing classes, trainers, membership plans, and member success stories. Designed to attract fitness enthusiasts and encourage gym memberships.

## Features
- **Class Listings**: 6 different fitness classes with schedules and member counts
- **Class Selection**: Interactive class details viewer
- **Trainer Profiles**: 4 certified expert trainers with specialties
- **Membership Plans**: 3-tier pricing (Basic, Premium, Elite)
- **Member Testimonials**: Success stories and transformations
- **Contact Form**: Lead generation for gym inquiries
- **Statistics**: Key metrics showcasing gym success
- **Responsive Design**: Mobile-friendly layout
- **Professional UI**: Modern fitness-focused design
- **Font Awesome Icons**: Professional icon library

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, flexbox, gradients, animations
- **JavaScript (ES6+)**: Class selection, pricing selection, form handling
- **Font Awesome 6.4**: Professional icon library (CDN)
- **No Dependencies**: Pure vanilla JavaScript
