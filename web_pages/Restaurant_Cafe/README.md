## Project Description
A professional restaurant website for Bella Cucina, an authentic Italian restaurant. Features interactive menu with filtering, online reservations, testimonials, and contact information with a sophisticated design.

## Features
- **Interactive Menu**: Browse menu items by category (Appetizers, Mains, Pasta, Desserts, Beverages)
- **Online Reservations**: Book a table directly through the website
- **Quick Info**: Location, hours, phone, and contact information
- **About Section**: Restaurant history and awards
- **Testimonials**: Customer reviews and ratings
- **Contact & Map**: Location details and social media links
- **Responsive Design**: Mobile-friendly layout
- **Professional UI**: Elegant design with gold accents
- **Font Awesome Icons**: Professional icon library

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, flexbox, gradients, animations
- **JavaScript (ES6+)**: Menu filtering, form handling, smooth scrolling
- **Font Awesome 6.4**: Professional icon library (CDN)
- **No Dependencies**: Pure vanilla JavaScript

## How to Use
1. Open `index.html` in a web browser
2. Browse the interactive menu by clicking on categories
3. View menu items with prices and descriptions
4. Make a reservation through the form
5. View customer testimonials
6. Contact the restaurant via phone or email
7. Visit social media links

## Menu Categories
- **Appetizers**: Bruschetta, Calamari, Caprese, Arancini
- **Main Courses**: Osso Buco, Branzino, Risotto, Saltimbocca
- **Pasta**: Carbonara, Pappardelle, Ravioli, Linguine
- **Desserts**: Tiramisu, Panna Cotta, Cannoli, Gelato
- **Beverages**: Wine, Prosecco, Espresso, Liqueur

## Customization
- Update menu items in the `menuData` object in `js/script.js`
- Modify restaurant information (address, phone, hours)
- Add real photo or map integration
- Update social media links
- Customize colors in CSS variables
