# Real Estate Agency Website - Premier Properties

## Project Description
A professional real estate agency website showcasing properties, featuring property search and filtering, agent profiles, services, and contact functionality. Designed to attract buyers, sellers, and investors.

## Features
- **Property Listing**: Browse 6+ properties with filtering by type
- **Search Form**: Location and property type search functionality
- **Property Details**: Beds, baths, square footage, and pricing
- **Agent Profiles**: 4 professional agents with ratings and contact info
- **Services Section**: 6 real estate services offered
- **Statistics**: Key metrics showcasing agency success
- **Virtual Tours Ready**: Prepared for 360Â° property tours
- **Contact Form**: Lead generation and inquiry submission
- **Responsive Design**: Mobile-friendly layout
- **Professional UI**: Modern design with teal and orange accents
- **Font Awesome Icons**: Professional icon library

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, flexbox, gradients, animations
- **JavaScript (ES6+)**: Property filtering, form handling, smooth scrolling
- **Font Awesome 6.4**: Professional icon library (CDN)
- **No Dependencies**: Pure vanilla JavaScript

## How to Use
1. Open `index.html` in a web browser
2. Use search form to find properties
3. Browse featured properties with filtering
4. Click on agents to view profiles and contact info
5. Review services offered
6. Fill out contact form for inquiries
7. Navigate using sticky navigation header
