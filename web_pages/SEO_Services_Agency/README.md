# SEO Services Agency Website - RankBoost SEO

## Project Description
A professional SEO services agency website showcasing comprehensive SEO solutions, proven results, and case studies. Features interactive pricing plans, service details, process explanation, and a contact form for lead generation.

## Features
- **Service Showcase**: 6 core SEO services with detailed descriptions
- **Process Timeline**: 4-step SEO process visualization
- **Case Studies**: 3 real case studies with measurable results
- **Pricing Plans**: 3-tier pricing structure with plan selection
- **Statistics**: Key metrics showing agency success
- **Contact Form**: Lead generation with industry selection
- **Responsive Design**: Mobile-friendly layout
- **Professional UI**: Modern design with blue and orange accents
- **Font Awesome Icons**: Professional icon library

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, flexbox, gradients, animations
- **JavaScript (ES6+)**: Plan selection, form handling, smooth scrolling
- **Font Awesome 6.4**: Professional icon library (CDN)
- **No Dependencies**: Pure vanilla JavaScript
